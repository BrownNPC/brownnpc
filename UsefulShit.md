## GameDev Guides
- [Import Minecraft worlds into Blender](https://m.youtube.com/shorts/NQVvH0EcPOo)
- [Quick Minecraft Blender Render Tutorial](https://www.youtube.com/watch?v=6FS98iDEjow)
- Import minecraft structures into blockbench
    1. Export your structure to `.nbt` using a Structure Block.
    2. Install the structure importer plugin in BlockBench.
    3. Extract minecraft.jar
    4. Make a new generic model in blockbench, then go to `file`, `import`.
    5. Point it to your extracted minecraft.jar's assets folder.
    6. Confirm, then import your .nbt
